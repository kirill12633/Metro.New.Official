Забыли пароль?

Для доступа к системе требуется авторизация через Active Directory

Требования безопасности:

  • Пароль должен содержать минимум 8 символов
  • Используйте комбинацию букв, цифр и специальных символов
  • Сессия автоматически завершится через 30 минут бездействия
  • Все действия логируются в системе аудита
